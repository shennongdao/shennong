# 文档目录

本目录是神农书库站点发布文档的源目录，归属于 `apps/web` 前端层。

适合放在这里的内容：

- 项目介绍；
- 架构设计；
- 用户指南；
- 协议规则；
- DAO 治理规则；
- 数据提供商规则；
- 网关运行说明；
- 爬虫运行说明；
- 开发文档；
- 安全与审计说明。

这些文档面向用户、开发者、数据提供商和治理参与者，应能被前端文档入口发布、被网关索引，并在正式阶段逐步迁移到 Arweave 长期访问。

不适合放在这里的内容：

- 个人调试笔记；
- 临时调研记录；
- 不准备公开发布的工程备忘；
- 与站点文档无关的仓库内部流程说明。

内部协作和临时记录应放在更合适的位置，例如根目录文档、对应模块的 `README.md`，或按个人偏好记录到 auto memory。
